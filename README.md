# Student Database Management System

## Overview
This project is a command-line based student database system built using Python and SQLite.

## Features
- Add student records
- View all students
- Update student details
- Delete student records
- Input validation and error handling

## Tech Used
- Python
- SQLite

## Concepts Used
- CRUD Operations (Create, Read, Update, Delete)
- Database management using SQL
- Modular programming

## How It Works
The system connects to a SQLite database and allows users to perform operations on student records through a menu-driven interface.
